## Radius, motion & type

- **Radius** — `--nx-radius-sm` · `--nx-radius` · `--nx-radius-lg`.
- **Motion** — `--nx-motion-fast|base|slow` + `--nx-ease-out|standard`. Standard transitions
  reference these; the canvas adds a reviewable preset layer (Smooth / Floaty / Snappy) on top.
- **Type** — `--nx-font-sans` (UI) and `--nx-font-mono` (hashes / indicators).

See the live, interactive versions in the **[Storybook → Foundations](/nexus/storybook/?path=/story/foundations-tokens--colors)** section.
