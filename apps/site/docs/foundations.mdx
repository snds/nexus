# Foundations

Tokens are the single source of truth, in three tiers: **L0 primitives** (raw ramps + the type
scale), **L1 semantics** (purpose aliases that point at primitives), and component tokens (with
their components). No component hardcodes a hue, size, or duration. Values are **read live from
`:root`**.

## Primitive color ramps (L0)

The raw scales every semantic color aliases. Neutral drives surfaces, borders and text; accent
drives interactive.

**Neutral** — `--nx-neutral-1 … 12` (1 = app bg → 12 = primary text)

<Ramp prefix="--nx-neutral" />

**Accent** — `--nx-accent-1 … 12` (9 = solid interactive)

<Ramp prefix="--nx-accent" />

## Type primitives (L0)

The type scale, weights, line-heights and tracking. Composed into the semantic text variants
below — never used as raw values in product.

<div style={{ display: 'flex', flexDirection: 'column', gap: 10, margin: '16px 0' }}>
  {['2xs','xs','sm','base','md','lg','xl','2xl','3xl','4xl','5xl'].map((s) => (
    <div key={s} style={{ display: 'flex', alignItems: 'baseline', gap: 16 }}>
      <code style={{ width: 120, color: 'hsl(var(--nx-fg-subtle))', fontSize: 11 }}>--nx-text-{s}</code>
      <span style={{ fontSize: `var(--nx-text-${s})`, color: 'hsl(var(--nx-fg))' }}>The quick brown fox</span>
    </div>
  ))}
</div>

- **Weights** — `--nx-weight-regular | medium | semibold | bold` (400 / 500 / 600 / 700)
- **Line height** — `--nx-leading-none | tight | snug | normal | relaxed`
- **Tracking** — `--nx-tracking-tight | normal | wide | wider`

## Text variants (semantic)

The `Text` component selects a semantic style composed from the type primitives. This is how
text is used in product — never raw `font-size`.

<div style={{ display: 'flex', flexDirection: 'column', gap: 12, margin: '16px 0' }}>
  {['display','h1','h2','h3','title','body','body-sm','label','caption','overline','code'].map((v) => (
    <div key={v} style={{ display: 'flex', alignItems: 'baseline', gap: 16 }}>
      <code style={{ width: 84, flexShrink: 0, color: 'hsl(var(--nx-fg-subtle))', fontSize: 11 }}>{v}</code>
      <Text variant={v}>The quick brown fox</Text>
    </div>
  ))}
</div>

Tones: <Text as="span" tone="default">default</Text> · <Text as="span" tone="muted">muted</Text> · <Text as="span" tone="subtle">subtle</Text> · <Text as="span" tone="accent">accent</Text>.
