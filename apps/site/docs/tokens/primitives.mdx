# Primitives (L0)

The raw scales every semantic token aliases. These are documented in full — not just the values
in use — so the whole range is visible. Values are read **live** from `:root`.

## Color ramps

Neutral drives surfaces, borders, and text; accent drives everything interactive. Each is a
12-step scale (1 = app background → 12 = primary text / solid accent).

**Neutral** — `--nx-neutral-1 … 12`

<Ramp prefix="--nx-neutral" />

**Accent** — `--nx-accent-1 … 12` (9 = solid interactive)

<Ramp prefix="--nx-accent" />

## Type scale

The raw sizes, composed into the semantic [text variants](/docs/foundations/typography) — never
used as raw values in product.

- **Weights** — `--nx-weight-regular | medium | semibold | bold` (400 / 500 / 600 / 700)
- **Line height** — `--nx-leading-none | tight | snug | normal | relaxed`
- **Tracking** — `--nx-tracking-tight | normal | wide | wider`

## Domain palettes

Two product-specific scales that sit alongside neutral/accent: the **entity** channel
(`--entity-*`) and the **severity** channel (`--severity-*`). They're documented on
**[Semantics](/docs/tokens/semantics)** with their swatches.
