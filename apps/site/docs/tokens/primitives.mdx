# Primitives (L0)

The raw scales every semantic token aliases. These are documented in full — not just the values
in use — so the whole range is visible. Values are read **live** from `:root`.

## Color ramps

Neutral is **Radix Slate**, accent is **Radix Indigo** — real Radix Colors scales, not hand-rolled
ramps. Each is a 12-step scale following the [step contract](/docs/tokens/token-model): 1–2 bg,
3–5 states, 6–8 borders, 9–10 solids, 11–12 text. Primitives are flat constants with **both**
light and dark values (`--nx-neutral-light-*` / `--nx-neutral-dark-*`); the dark set is shown here.

**Neutral · Slate** — `--nx-neutral-dark-1 … 12`

<Ramp prefix="--nx-neutral-dark" />

**Accent · Indigo** — `--nx-accent-dark-1 … 12` (9 = solid · 11 = accent text)

<Ramp prefix="--nx-accent-dark" />

## The full Radix palette

Every Radix scale (25 colors + 6 grays) ships in the system as `--nx-<scale>-<light|dark>-<1…12>`,
so any hue stack is readily available across the UI — and every entity, severity, and context
channel maps onto one of these (see [Semantics](/docs/tokens/semantics)). Dark steps shown.

      <Ramp prefix={`--nx-${s}-dark`} />

</div>

## Type scale

The raw sizes, composed into the semantic [text variants](/docs/foundations/typography) — never
used as raw values in product.

    <div key={s} style={{ display: 'flex', alignItems: 'baseline', gap: 16 }}>
      <code style={{ width: 120, color: 'var(--nx-fg-subtle)', fontSize: 11 }}>--nx-text-{s}</code>
      <span style={{ fontSize: `var(--nx-text-${s})`, color: 'var(--nx-fg)' }}>The quick brown fox</span>
    </div>

- **Weights** — `--nx-weight-regular | medium | semibold | bold` (400 / 500 / 600 / 700)
- **Line height** — `--nx-leading-none | tight | snug | normal | relaxed`
- **Tracking** — `--nx-tracking-tight | normal | wide | wider`

## Domain palettes

Two product-specific scales that sit alongside neutral/accent: the **entity** channel
(`--entity-*`) and the **severity** channel (`--severity-*`). They're documented on
**[Semantics](/docs/tokens/semantics)** with their swatches.
