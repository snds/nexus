# Semantics (L1)

Purpose aliases that point at the [primitive ramps](/docs/tokens/primitives). Components consume
**these**, never the raw scales — so re-pointing a primitive in one theme updates everything.

## Channels map to Radix hue stacks

Entity, severity, and context are **semantic channels**, each assigned a full **Radix hue stack** —
not a one-off color. The single token (e.g. `--severity-malicious`) resolves to its hue's solid
**step 9**; the rest of that hue's stack is reachable via `--nx-<hue>-<light|dark>-<step>` for bg
(step 3), border (6), or text (11) variants. Because they're semantics, they flip with the theme.

- **Motion** — `--nx-motion-fast | base | slow` + `--nx-ease-out | standard`. See
