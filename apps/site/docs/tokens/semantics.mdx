# Semantics (L1)

Purpose aliases that point at the [primitive ramps](/docs/tokens/primitives). Components consume
**these**, never the raw scales — so re-pointing a primitive in one theme updates everything.

## Radius & motion

- **Radius** — `--nx-radius-sm` · `--nx-radius` · `--nx-radius-lg`.
- **Motion** — `--nx-motion-fast | base | slow` + `--nx-ease-out | standard`. See
  **[Foundations → Motion](/docs/foundations/motion)**.
