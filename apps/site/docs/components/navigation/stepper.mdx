# Stepper

## Usage

Use a stepper to show position in a linear, finite flow (3–5 steps). Pair it with the content for
the current step; the stepper itself reports progress, it doesn't navigate.

**When not to use:** for open-ended navigation, use the nav rail. For a single yes/no, use a
**[Modal](/docs/components/overlays/modal)**.

## Example

<Specimen>
  <Stepper steps={['Connect','Authorize','Verify']} current={1} />
</Specimen>

## Accessibility

- Steps expose their order and completed/current state to assistive tech.
- Don't rely on color alone for the current step — position and label carry it too.
