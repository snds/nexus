## Usage

Use `EntityIcon` to render a threat entity's glyph with its canonical type color applied — actor,
campaign, malware, domain, email, and so on. This is how color = type is enforced consistently.

**When not to use:** for a generic UI glyph with no entity meaning, use
**[Icon](/docs/components/iconography/icon)**.

## Accessibility

- Color encodes type but is never the only signal — entity icons appear with a label in context
  (node, legend, row).
