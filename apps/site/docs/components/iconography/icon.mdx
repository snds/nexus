## Usage

Use `Icon` for any glyph. Symbols are **outlined by default**; flip `filled` only for active /
selected / hub states (see the [fill principle](/docs/foundations/principles)). Never go below the
20 px minimum legible size.

**When not to use:** for an entity type glyph in the graph, use
**[EntityIcon](/docs/components/iconography/entity-icon)** so the color channel is applied for you.

## Accessibility

- Decorative icons are hidden from assistive tech; pass `title` to give a meaningful icon an
  accessible name.
- Icon-only controls must carry a label via their control (e.g. `ToolButton`), not the icon alone.
