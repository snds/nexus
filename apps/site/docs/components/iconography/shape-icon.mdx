## Usage

Use `ShapeIcon` to show the **family** channel — the node silhouette that pairs with color (type)
and the verdict pip. It backs the graph legend and documents node anatomy.

**When not to use:** to render a full node, use **[NodeBadge](/docs/components/graph/node-badge)**.
