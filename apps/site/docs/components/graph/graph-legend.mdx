# GraphLegend

## Usage

Use `GraphLegend` to give the canvas a key for its two color channels — entity type and verdict
severity. It is **collapsed by default** so it never competes with the graph, and uses bare,
larger icons (no shape swatches) for the type rows.

**When not to use:** outside the graph. For a single label, use a
**[Pill](/docs/components/data-display/pill)** or **[VerdictBadge](/docs/components/data-display/verdict-badge)**.

## Example

<Specimen padded={false}>
  <GraphLegend
    types={['actor','campaign','malware','domain','email']}
    verdicts={['malicious','suspicious','benign']}
    defaultCollapsed={false}
  />
</Specimen>

## Accessibility

- Each row pairs the color/icon with its text label — the legend itself is the proof that color is
  never the sole carrier.
- The collapse control is a labelled, keyboard-operable button.
