# The node system

`NodeBadge` is the variant-driven graph node — the Figma "Node" component set. **Color = family,
shape = family macro, icon = the specific type, verdict = its own pip**; flags (VIP / status /
imposter / +N) combine into one top-right cluster. Every node shares a translucent token fill
behind a backdrop blur so edges recede behind it.

A node's **shape and color both come from one source** — the entity's family (`ENTITY_FAMILIES`
in `@nexus/domain`). The [GraphLegend](/docs/components/graph/graph-legend) reads the *same* source,
so the legend and the canvas always match. Shape is the coarse macro (circle threats · square
infrastructure · diamond detections/identifiers · hexagon comms); color is the specific
[family](/docs/foundations/color); the icon is the type within it.
