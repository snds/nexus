# The node system

`NodeBadge` is the variant-driven graph node — the Figma "Node" component set. **Color = type,
shape = family, verdict = its own pip**; flags (VIP / status / imposter / +N) combine into one
top-right cluster. Every node shares a translucent token fill behind a backdrop blur so edges
recede behind it.
