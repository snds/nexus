## Usage

Use a pill for a short, non-interactive label — an entity type, a tag, a category. Keep the text
to one or two words.

**When not to use:** for a threat verdict, use **[VerdictBadge](/docs/components/data-display/verdict-badge)**
(its own semantic channel). For a numeric overflow count, use
**[CountChip](/docs/components/data-display/count-chip)**.

## Tones

<Specimen>
  <Pill tone="neutral">neutral</Pill>
  <Pill tone="accent">accent</Pill>
</Specimen>

## Accessibility

- Renders inline text — contrast meets AA in both themes via the semantic tokens.
- Carries no interactive semantics; don't attach click handlers (use a Button).
