# StatCircle

<ComponentMeta
  status="stable"

  storybook="?path=/docs/components-data-display-statcircle--docs"

## Usage

Use a stat circle for a single key metric with a short label — DLP events, login counts, clean
rate. Tone the circle to carry severity where relevant.

**When not to use:** for a plain inline number, use **[CountChip](/docs/components/data-display/count-chip)**.
For multiple related metrics, lay several stat circles in a row.

## Tones

<Specimen>
  <StatCircle label="DLP" value={42} tone="malicious" />
  <StatCircle label="Logins" value={1280} tone="neutral" />
  <StatCircle label="Clean" value={98} tone="benign" />
</Specimen>

## Accessibility

- The label and value are both text; tone color is reinforcement, not the sole signal.
