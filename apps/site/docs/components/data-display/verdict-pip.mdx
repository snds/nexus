## Usage

Use a verdict pip where space won't allow a labelled badge — on a `NodeBadge`, in a tight table.
It carries the same `--severity-*` channel as the badge.

**When not to use:** standalone in prose or a panel where a reader needs the word — use
**[VerdictBadge](/docs/components/data-display/verdict-badge)**.

## Accessibility

- Because the pip has no visible label, always pair it with text nearby (the node label, a row
  cell) or an accessible name so meaning isn't color-only.
- `ringColor` lets the pip sit legibly on any node fill.
