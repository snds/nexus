## Usage

Use a count chip to surface a number that represents more — collapsed children on a node, items
beyond a visible limit. Optionally prefix it (e.g. `+`).

**When not to use:** for a labelled metric, use **[StatCircle](/docs/components/data-display/stat-circle)**.

## Examples

<Specimen>
  <CountChip count={3} />
  <CountChip count={12} prefix="+" />
  <CountChip count={148} prefix="+" />
</Specimen>

## Accessibility

- Numeric content reads naturally; pair with context (the node label) so the number's meaning is clear.
