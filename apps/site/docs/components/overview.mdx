# Components

Every component has its own page — anatomy, usage, states, accessibility, and API — under the
categories in the sidebar. Each page embeds the real `@nexus/ui` primitive and deep-links to its
**[Storybook](/nexus/storybook/)** playground (full variant matrix + the a11y addon).

The library has **two tiers**: **L1 base** are pristine shadcn/Radix primitives (Button, Tooltip,
Dialog, DropdownMenu, Popover); **L3 product primitives** compose them with tokens. What follows is
a visual index — click into a category for the documented page.
