# Packages & assets

## Code packages

| Package | Import | Contains |
|---|---|---|
| `@nexus/tokens` | `@nexus/tokens/tokens.css` | The `--nx-*` / `--entity-*` / `--severity-*` source of truth + the shadcn bridge. |
| `@nexus/ui` | `@nexus/ui/nexus` | L1 shadcn/Radix bases + L3 product primitives. |

Install and use them from **[Get started → For developers](/docs/get-started/for-developers)**.

## The three surfaces

- **Documentation** — this site (foundations, tokens, components, patterns, content).
- **[Storybook](/nexus/storybook/)** — the interactive component explorer with autodocs + the
  accessibility addon.
- **[Live demo](/nexus/app/)** — the Threat Explorer, the system in real product use.
